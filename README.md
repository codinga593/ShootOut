# ShootOut
A multiplayer LAN 2D Shooter Game. 

Note: Server and Client Must be Run on different machines or on different python instances on the same machine.

IP Address on the client must match the server for connection. The server can be set to 0.0.0.0 to listen on all interfaces if required. Client and Server must be on same local network for multiplayer games. (LAN only)

The whole folder must be unpacked and directories unchanged for sound files and assets. In case assets are not found, player and gun assets will default to simple shapes. 

For an easy download, just download the zip rather than cloning the repository.
